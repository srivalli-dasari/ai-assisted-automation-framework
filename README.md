# AI Assisted Automation Framework

## Overview
This project is a beginner-friendly, real-world style automation framework
created to learn and demonstrate the skills required for
SDET / AI Test Engineer roles.

The framework will cover:
- UI automation testing
- API testing
- CI/CD integration
- AI-assisted testing concepts (future scope)

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- REST Assured
- GitHub Actions
